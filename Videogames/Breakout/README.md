#Gameplay
A/Left Arrow - Move the paddle left
D/Right Arrow - Move the paddle right
Space - Start the game whilst the ball is paused by shooting it off upwards towards a random direction

#Objective
Clear all the blocks on the screen before running out of lives. If you run out of lives you must refresh the page to play again. If you clear all the blocks you win, but you must also refresh the page to play again.

#Twist
After every 16th point the controlers invert, so after the firts 16 points

A/Left Arrow - Move the paddle right
D/Right Arrow - Move the paddle left

There will be a text on screen explaining this.
